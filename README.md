### üìò Description
This Java program is a simple GUI (Graphical User Interface) application that shows product sales data for a local sound equipment retailer.

It displays:
- Total Sales  
- Average Sales  
- Number of Sales Over the Limit  
- Number of Sales Under the Limit  
- Years Processed  

The current sales limit is **500**.

---

### üñ•Ô∏è Features
1. **Load Product Data**  
   Loads product sales from memory and shows the total, average, and other statistics in a text area.

2. **Save Product Data**  
   Saves the displayed data to a file called `data.txt`.

3. **Clear**  
   Clears the text area and the label showing years processed.

4. **Exit**  
   Closes the program.

5. **Menu Bar**  
   The program includes a menu with *File* and *Tools* options to perform the same actions as the buttons.
